DOM stands for "Document Object Model." It is a programming interface and representation of a structured document, typically used for web pages and XML documents. The DOM represents the structure of the document as a tree of objects, where each object corresponds to a part of the document, such as elements, attributes, and text content.In the context of web development, the DOM is often associated with web browsers. When a web page is loaded, the browser creates a DOM representation of the HTML or XML content of the page. This allows developers to interact with and manipulate the content of the page using scripting languages like JavaScript. By accessing and modifying the DOM, developers can dynamically update the content and appearance of a web page without requiring a full page reload.

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.

A JavaScript callback is a function which is to be executed after another function has finished execution. A more formal definition would be - Any function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function.
